disba.depthplot(x, z, zmax, ax=None, **kwargs)[source]

Vertical step plot.

Parameters

  • x (array_like) – X coordinates of data points.

  • z (array_like) – Z coordinates of data points.

  • zmax (scalar) – Depth of last data point.

  • ax (matplotlib.pyplot.Axes or None, optional, default None) – Matplotlib axes. If None, a new figure and axe is created.

Returns

Plotted data axes.

Return type

matplotlib.pyplot.Axes
